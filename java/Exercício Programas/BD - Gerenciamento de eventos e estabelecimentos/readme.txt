=========================
EXECUÇÃO DO APLICATIVO
=========================
Para executar o aplicativo, execute o seguinte 
comando no diretório JAR Executável dentro desta pasta:

execute o .bat dentro de "./Agenda/src/" de nome Compilar - Executar.bat
ou
java -jar "Agenda.jar" dentro de "./Agenda/dist/"
ou
Abra pelo IDE Netbeans
=========================
CONSTRUÇÃO
=========================

Foi implementado utilizando a IDE NetBeans 7.3 com JDK 7.0,
no entanto a compatibilidade com a versão

anterior do JDK 6.0
