Para compilar:

No windows:
	1.Pode-se usar o Dev c++, simplesmente criar um projeto e clicar em compilar
	ou compilar e depois executar.

	2.Pode-se também compilar por outro programa deste que o mesmo use o gcc e as
	biliotecas do mesmo.

	3.Ou compilar escrevendo 
		>>gcc -o main main.c -lm<<
		no prompt. (deste que o gcc esteja devidamente instalado).

No linux:
	1. Digitar 
		>>gcc -o main main.c -lm<<
		no terminal.

Para rodar:
	Digitar:
	>>>./main entrada.txt saida.txt

onde entrada.txt pode ser substituido por algum outro arquivo de entrada.
e saida.txt pode ser substituido para outro arquivo de saida
