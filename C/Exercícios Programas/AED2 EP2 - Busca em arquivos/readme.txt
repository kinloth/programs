 - Instruções de Compilação:
	Para a compilação do programa executar o comando:
		"gcc principal.c create_data.c buscas.c -o principal "
		
 - Instruções de Execução:
	Para a execução do programa, basta executar o programa principal, da forma:
		"./principal "
	sendo que o arquivo "config.txt" deverá estar escrito no formato:
		<operação>
		<arquivos de entrada>
		<arquivos de saída>
		<número de buscas>
		<id 0>
		<id 1>
		<id 2>
		...
		<id k-1>
